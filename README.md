
## Cómo Usar el Panel de Administración

### 1. Acceder al Panel
- Ve a `admin.html` desde el enlace en el menú principal
- O visita directamente: `tu-sitio.com/admin.html`

### 2. Agregar Productos
1. Completa el formulario "Agregar Nuevo Producto"
2. Selecciona una imagen del producto
3. Elige la categoría y marca
4. Asigna una etiqueta (Nuevo, Oferta, etc.)
5. Haz clic en "Agregar Producto"

### 3. Gestionar Productos Existentes
- **Editar**: Haz clic en "Editar" en cualquier producto
- **Eliminar**: Haz clic en "Eliminar" para borrar un producto
- **Exportar**: Descarga todos los productos en formato JSON

### 4. Configurar Producto Destacado
- Selecciona un producto para mostrar en la portada
- Haz clic en "Actualizar Portada"

## Notas Importantes

- Las imágenes se guardan localmente (en un entorno real deberías subirlas a un servidor)
- Todos los datos se guardan en el localStorage del navegador
- El panel admin es accesible desde el enlace en el menú principal
